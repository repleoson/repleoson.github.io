This short demo shows you how to easily get started with the Amazon Elastic File System Infrequent Access (EFS IA) storage class and optimizing storage cost with a EFS Lifecycle Management policy.

Amazon EFS Infrequent Access provides price and performance for files not accessed everyday and automatically moves your files to the lower cost EFS IA storage class based on the lifecycle management policy you choose.
