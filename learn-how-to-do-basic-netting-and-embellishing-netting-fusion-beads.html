In this video you will learn how to do basic beaded netting using size 11 seed beads. Then we’ll show you how to embellish the beaded netting using size 15 seed beads and 4mm Swarovski Crystal bicones. This is a fun technique that you can easily adjust. Change up the width and the colors of the embellishment.
