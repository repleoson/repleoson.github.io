Before You Install Linux - STOP! Ask Yourself Some Questions First

Linux is a great operating system with many solid features, custom options and it’s very secure. But it is it for everyone?

The answer is straightforward. NO! If you’re a technically knowledgeable person you may well find the whole process easy, but for the uninitiated it could be a frustrating and difficult experience

The key is making sure you do plenty of pre-installation research, enjoy the learning process, and don’t be afraid to try new things, including the terminal

If you rely on proprietary software you may decide that Linux isn’t for you. This is a legitimate reason for staying where you are because Wine, the software that can run some Windows programmes on Linux, is far from perfect

Overall this video just gives you a heads up about some of things you should think about before taking the leap. And for existing Linux users who want to evangelise and install Linux on everyone’s computers, it highlights some of the risks of doing that
