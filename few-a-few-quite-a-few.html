In this video, learn the difference between (few), (a few ) and (quite a few) & how to use them in English.

We use both «few» and «a few» to talk about more than one thing but not a huge amount. However, there is a big difference between them.

"Few " means not many.
"A few" means some.
For example, I have few friends = I don’t have many friends.
I have a few friends = I have some friends.

"Quite a few" talks about a huge amount and it means a lot of.
For example, I have quite a few friends = I have a lot of friends.
