In this Diablo 2 Resurrected Best Class Guide we are going to be talking about the Best Diablo 2 Classes, how to play each of them, and which are the right one for you. The first thing that you’ll need to do when you start the game is decide which Class you’ll be playing. If you are a newcomer I strongly recommend that you pick a class based on your liking and not based on a build guide. The reason for this, is that most builds heavily rely on powerful items that you’ll likely won’t be able to get for at least the first 50 to 100 hours.
