Can and could are used to talk about abilities in the past, present or future. Sometimes, there isnâ€™t a form of can, and we need to use the verb be able to. For example, can doesnâ€™t have a present perfect form, so if you want to use can in the present perfect, you need to use be able to.

In other forms, can or could and be able to generally have the same meaning. However, if you are talking about abilities in the past, could and be able to are sometimes different.
