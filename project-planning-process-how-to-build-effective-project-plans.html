The project planning process can be as complex as you need it to be or as simple as you want it to be. Each step can be detailed or summary. And you need only use the steps that are right for your project.

All the steps in the project planning process work together. But in this video, project management expert, Dr Mike Clayton, ties them together in a logical sequence. It will help you understand the purpose of each of the project planning tools and how they connect to one another.
