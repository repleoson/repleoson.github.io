This tutorial will show you how to use the recently released Bulletins Sample App in Microsoft Teams. The Microsoft Teams Bulletins app allows organizations to share and organize content easily within Teams. Specifically, the Microsoft Teams Bulletins app allows for the sharing of articles or posts, frequently asked questions (FAQs), and key links and contacts. In the Bulletins app, all of these information elements can be organized into customizable categories. The Microsoft Teams Bulletins app also allows users to view analytics for the content created and shared in the app.
