The B5600 dispensing machine is used to glue electronic components to a circuit board to secure them and to protect them against vibrations. The fully automatic dispensing cell is perfectly suited for in-line integration. Thanks to the 4-axis gantry system with 45° swivel unit and 90° work piece carrier rotation, it is a compact and economical alternative for a 6-axis robot solution.

The dispensing application is done with bdtronic’s own eccentric screw pumps. The silicone can be applied with extreme precision on both the top and bottom of the component.
